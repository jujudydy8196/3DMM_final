This package contains the implementation of the defocus estimation
algorithm presented in the paper:

"Recovering Depth from a Single Defocued Image", 
Shaojie Zhuo and Terence Sim, Pattern Recognition, 2011

Use of this code is permitted only for academic purpose. If you 
generate data using the code for an academic publication, please 
include the following citation in your paper. 
  
@article{ZhuoAndSim_2011,
  author    = {Shaojie Zhuo and Terence Sim},
  title     = {Defocus map estimation from a single image},
  journal   = {Pattern Recognition},
  volume    = {44},
  number    = {9},
  year      = {2011},
  pages     = {1852-1858}
}

If you want to use the code for other purposes, please contact us. 

The code has been tested on Matlab 7.14 (R2012a) on Windows 7.
A few test images have also been provided. 

To report any bugs, please email: Shaojie Zhuo, jayzhuo@gmail.com
